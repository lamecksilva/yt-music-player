# yt-music-player

A CLI music player with Node.js

Usage:

```js
/bin/bash
$ node index.js https://www.youtube.com/watch?v=ZVao4az7kgM
```

```js
/bin/fish
$ node index.js 'https://www.youtube.com/watch?v=ZVao4az7kgM'
```
